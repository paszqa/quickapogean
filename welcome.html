Welcome! This website is dedicated to the game Apogea, a really cool MMORPG. The game is currently in development, time-limited playtests happen from time to time, and the game is expected to be released in 2025.

Currently, there is no playtest happening, but you can check the official Discord for more information. The astrologists are predicting the next playtest might be opened some time between April and June, so stay tuned.

There are no quest spoilers on this site, and there won't be any. There are lists of items, spells, calculators, and other references that may prove useful when deciding on a build or choosing which path to take.
